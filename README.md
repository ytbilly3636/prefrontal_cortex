# prefrontal_cortex

## What is this
This repository is for developing reservoir computing models (echo state networks).  
Especially, the models are used for prefrontal cortex models.

## Requirement
* Python
* NumPy
* Matplotlib

## Quick Start
To execute test script of echo state network,

```sh
$ python esn.py
```
